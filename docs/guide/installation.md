# Installation

## Prerequisites

- Node.js 18+ 
- npm, yarn, or pnpm

## Install

### Using npm
```bash
npm install rnode-server
```

### Using pnpm
```bash
pnpm add rnode-server
```

### Using yarn
```bash
yarn add rnode-server
```

## Verify Installation

```bash
node -e "console.log(require('rnode-server'))"
```

## Platform Support

RNode Server supports multiple platforms:

- **macOS**: arm64, x64
- **Linux**: arm64, x64, arm-gnueabihf
- **Windows**: arm64, x64
- **Android**: arm-eabi
